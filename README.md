## Styling

The project uses a custom Tailwind CSS configuration with the following brand colors:

```javascript
colors: {
  altora: {
    navy: "#0A2342",
    gold: "#C6A760",
    "gold-light": "#E5D5A7",
    gray: "#F5F5F5",
    "gray-dark": "#4A4A4A",
  }
}
```

Typography:
- Headings: Playfair Display (serif)
- Body: Inter (sans-serif)
